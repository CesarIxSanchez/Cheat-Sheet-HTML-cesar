Glosario de HTML

Alumno: César Antonio Ix Sánchez. Matricula: 66271

<!DOCTYPE html>

Indica al navegador que el documento está escrito en HTML5. No es una etiqueta, sino una declaración que ayuda a interpretar correctamente el código.

Ejemplo en código:

<!DOCTYPE html>

<html>

Es la etiqueta raíz que encierra todo el contenido de la página web. Todo el documento va dentro de esta.

Ejemplo en código:

<html lang="es">
<!-- Aquí dentro van <head> y <body> -->
</html>

<head>

Contiene información sobre el documento, como metadatos, enlaces a hojas de estilo, scripts o el título de la página. No muestra contenido directamente en la página.

Ejemplo en código:

<head>
  <meta charset="utf-8">
  <title>Mi primera página</title>
</head>

<body>

Encierra todo el contenido visible de la página: textos, imágenes, tablas, formularios, etc. Es lo que el usuario ve en el navegador.

Ejemplo en código:

<body>
  <h1>Hola Mundo</h1>
  <p>Este es el contenido visible.</p>
</body>

<title>

Define el título de la pestaña del navegador. Se escribe dentro del <head> y no aparece directamente en el cuerpo de la página.

Ejemplo en código:

<head>
  <title>Catálogo de Aguas</title>
</head>

<p>

Representa un párrafo de texto. Es una de las etiquetas más usadas para escribir contenido.

Ejemplo en código:

<p>Este es un párrafo de ejemplo en HTML.</p>

Este es un párrafo de ejemplo en HTML.

<header>

Representa la cabecera de un documento o sección. Normalmente contiene el logo, el título o la barra de navegación principal.

Ejemplo en código:

<header>
  <h1>Mi Sitio Web</h1>
  <p>Bienvenido a la página oficial.</p>
</header>

El ejemplo lo tienes en la cabecera del inicio de la página.

<nav>

Define un bloque de enlaces de navegación. Se utiliza para menús, barras de navegación o listas de enlaces internos.

Ejemplo en código:

<nav>
  <ul>
    <li><a href="index.html">Inicio</a></li>
    <li><a href="productos.html">Productos</a></li>
    <li><a href="contacto.html">Contacto</a></li>
  </ul>
</nav>

El ejemplo lo tienes a la izquierda de la página.

<main>

Encierra el contenido principal del documento. Solo debe haber un <main> por página y no debería contener cosas repetitivas como menús o pies de página.

Ejemplo en código:

<main>
  <h2>Catálogo de productos</h2>
  <p>Aquí van los artículos destacados.</p>
</main>

<aside>

Representa contenido complementario o secundario, como barras laterales, anuncios o notas adicionales.

Ejemplo en código:

<aside>
  <h3>Publicidad</h3>
  <p>Compra 2x1 en refrescos esta semana.</p>
</aside>

<footer>

Define el pie de página de un documento o sección. Normalmente contiene información de autor, enlaces de contacto, derechos de autor o mapas del sitio.

Ejemplo en código:

<footer>
  <p>&copy; 2025 Mi Sitio Web</p>
  <a href="aviso.html">Aviso de privacidad</a>
</footer>

El ejemplo lo tienes hasta el final de la página.

<h1> a <h6>

Definen encabezados jerárquicos en HTML. <h1> es el de mayor importancia (título principal), mientras que <h6> es el de menor relevancia. Se usan para estructurar el contenido y facilitar la lectura.

Ejemplo en código:

<h1>Encabezado principal</h1>
<h2>Subencabezado</h2>
<h3>Sección secundaria</h3>

Soy un encabezado h3

Soy un encabezado h4

<span>

Es un contenedor en línea que no tiene un significado semántico propio. Se usa para aplicar estilos o agrupar texto dentro de un párrafo sin alterar la estructura del documento.

Ejemplo en código:

<p>Este es un <span style="color: red;">texto resaltado</span> dentro de un párrafo.</p>

Nota: No se debe poner un "style" dentro de una etiqueta, esto es un ejemplo de uso, favor de no replicarlo.

<a>

Crea hipervínculos hacia otras páginas, archivos, secciones internas o correos electrónicos. Puede usarse tanto para navegación externa como interna.

Ejemplo en código:

<a href="https://www.ejemplo.com">Visitar sitio</a>

<ol>

Define una lista ordenada, donde los elementos aparecen numerados o con un marcador definido.

Ejemplo en código:

<ol>
  <li>Primer paso</li>
  <li>Segundo paso</li>
  <li>Tercer paso</li>
</ol>

  1. Primer paso
  2. Segundo paso
  3. Tercer paso

<ul>

Define una lista no ordenada, donde los elementos se muestran con viñetas.

Ejemplo en código:

<ul>
  <li>Manzanas</li>
  <li>Peras</li>
  <li>Plátanos</li>
</ul>

  • Manzanas
  • Peras
  • Plátanos

<li>

Representa un elemento dentro de una lista, ya sea ordenada (<ol>) o no ordenada (<ul>).

Ejemplo en código:

<ul>
  <li>Elemento de lista</li>
</ul>

  • Elemento de lista

<dl>

Crea una lista de definiciones, útil para glosarios o pares de término-descripción.

<dt>

Define un término dentro de una lista de definiciones (<dl>).

<dd>

Proporciona la descripción o definición de un término dentro de una lista de definiciones (<dl>).

Ejemplo en código:

<dl>
  <dt>HTML</dt>
  <dd>Lenguaje de marcado para crear páginas web.</dd>
</dl>

<table>

Representa una tabla de datos estructurada en filas y columnas.

<tr>

Define una fila dentro de una tabla.

<td>

Representa una celda de datos en una tabla.

<th>

Define una celda de encabezado dentro de una tabla. Normalmente se muestra en negrita y centrada.

<thead>

Agrupa el encabezado de una tabla, que contiene filas con títulos de columna.

<tbody>

Contiene el cuerpo principal de una tabla, donde se colocan los datos.

<tfoot>

Define el pie de una tabla, normalmente usado para totales o notas.

Ejemplo en código:

<table border="1">
  <thead>
    <tr>
      <th>Mes</th>
      <th>Ventas</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Enero</td>
      <td>$200</td>
    </tr>
    <tr>
      <td>Febrero</td>
      <td>$250</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>Total</td>
      <td>$450</td>
    </tr>
  </tfoot>
</table>

Mes Ventas
Enero $200
Febrero $250
Total $450

<img>

Inserta una imagen en el documento. Debe incluir el atributo src (fuente) y se recomienda alt (texto alternativo).

Ejemplo en código:

<img src="/assets/minion.jpg" alt="Imagen de un minion tierno">

<audio>

Reproduce archivos de audio directamente en la página. Puede incluir controles para pausar o ajustar volumen.

<source>

Define múltiples fuentes para <audio> o <video>, permitiendo compatibilidad con diferentes formatos.

Ejemplo en código:

<audio controls>
  <source src="/assets/himno-nacional.mp3" type="audio/mpeg">
</audio>

Favor de hacer clic en reproducir.

<video>

Inserta un archivo de video en el documento, con soporte para controles como reproducción o pantalla completa.

<source>

Define múltiples fuentes para <audio> o <video>, permitiendo compatibilidad con diferentes formatos.

Ejemplo en código:

<video controls width="300">
  <source src="/assets/clair-obscur.mp4" type="video/mp4">
</video>

<iframe>

Inserta una página web o recurso externo dentro de otra página.

Ejemplo en código:

<iframe src="https://www.wikipedia.org" width="600" height="500"></iframe>

<object>

Permite incrustar recursos externos como imágenes, PDFs o aplicaciones en una página web.

Ejemplo en código:

<object data="/assets/ejemplo.pdf" type="application/pdf" width="600" height="500"></object>

<form>

Define un formulario para recopilar datos del usuario y enviarlos a un servidor.

<input>

Campo de entrada para recibir datos del usuario, como texto, contraseñas, correos o botones.

<select>

Crea un menú desplegable con varias opciones.

<option>

Define cada opción dentro de un menú desplegable (<select>).

<textarea>

Crea un área de texto de varias líneas para entradas largas como comentarios o mensajes.

<label>

Proporciona una etiqueta descriptiva para un campo de formulario, mejorando la accesibilidad.

Ejemplo en código:

<form>
  <p>Formulario de ejemplo</p>
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre" placeholder="Escribe tu nombre">
  <label for="correo">Correo electrónico:</label>
  <input type="email" id="correo" name="correo" placeholder="Escribe tu correo">
  <label for="pais">País:</label>
  <select id="pais" name="pais">
    <option value="mx">México</option>
    <option value="es">España</option>
    <option value="ar">Argentina</option>
  </select>
  <label for="mensaje">Mensaje:</label>
  <textarea id="mensaje" name="mensaje" placeholder="Escribe tu mensaje aquí..."></textarea>
  <button type="button">Enviar</button>
</form>
